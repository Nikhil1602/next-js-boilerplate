## NextJs + Typescript - Boilerplate

![Thumbnail Image](./public/thumbnail.png)

### Setup includes:

- React 19
- Next 16
- Tailwind 4
- Typescript 5
- Font - Poppins
- Shadcn/UI - Neutral

### Commands to run the app

```bash
git clone https://github.com/Nikhil1602/next-js-boilerplate.git
cd next-js-boilerplate
npm run dev
```
